# ระบบสถานีตรวจวัดสภาพอากาศบน IOT

## Json Server Command
<code>json-server --watch data.json --host <input_ip> --port <input_port></code>

## โครงสร้างข้อมูล
```json
{
  "sensors": [
    {
      "message": "{\"humidity\":55.000000, \"temperature\":23.700001}",
      "id": 1
    },
    {
      "message": "{\"humidity\":95.000000, \"temperature\":27.200001}",
      "id": 2
    }
  ]
}
```
